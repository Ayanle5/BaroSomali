# Baro Somali

**Baro Somali** is an interactive web-based platform for learning the Somali language. It combines structured lessons, quizzes, a community chat, and a word bank to help learners go from beginner to fluent.

> "Baro" means *"Learn"* in Somali.

---

## ✨ Features

- 📘 **Structured Lessons** — Progressive lessons from beginner to advanced, with multiple question types including multiple choice and typed answers
- 💬 **Community Chat** — Real-time chat with other learners to practise Somali in conversation
- 📖 **Word Bank** — Searchable and filterable phrasebook of Somali vocabulary with English translations
- 📊 **Progress Tracking** — XP system, streaks, and lesson completion stats saved per user
- 🌙 **Dark Mode** — Full dark/light mode toggle
- 📱 **Mobile Responsive** — Optimised for all screen sizes
